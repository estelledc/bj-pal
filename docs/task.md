# 短时活动规划与执行 Agent · 任务说明

## 1. 任务背景

小明周六上午难得双休，他给美团发了一条消息：**「今天下午有空，想和老婆孩子 / 朋友们出去玩几个小时，别离家太远，帮我安排一下。」** 然后他**把手机递给老婆**，或**把安排发给朋友们**，看他们是否认可。

这不是传统「搜索推荐」，而是**「帮人把事做完」**：美团应在接下来几分钟内，把一下午安排成可落地的行动。

---

## 2. 典型场景与用户画像

| 场景       | 说明 |
| ---------- | ---- |
| **家庭**   | 孩子约 5 岁；配偶近期在减脂，饮食需兼顾。 |
| **朋友**   | 一共 4 人，2 男 2 女，偏好可能不同。 |

---

## 3. 产品期望能力

- **时间与动线**：规划「下午约 4–6 小时」的完整方案，例如：去哪玩 → 玩完去哪吃 → 是否有加餐/加餐活动。
- **餐厅与余位**：查到适合群体的餐厅，核验是否有座、是否要排队或可预约。
- **活动编排**：行前/餐后活动（亲子场所、展览、citywalk、小吃街等），与就餐时间衔接。
- **可执行闭环**：方案经用户确认后，能**一键**完成下单、预约，以及送至指定餐厅的蛋糕鲜花等配套服务。
- **对外沟通**：可把最终计划话术化分享给他人，例如：「搞定了，下午 2 点出发，先去……」发给朋友「小张」等。

---

## 4. 任务描述（技术方向）

构建一个**本地场景的短时活动规划与执行 Agent**：

- **输入**：一句自然语言目标（含时间、同伴、地点偏好等）。
- **输出**：完整、可执行的方案；在 Demo 范围内尽量模拟**自动完成关键下单 / 预订**。

交付形态：**Demo**（命令行或 Web UI 均可）。
